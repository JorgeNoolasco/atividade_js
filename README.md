# 🚀 Lista de Exercícios - JavaScript Básico

Este repositório contém a resolução de uma lista com 20 exercícios práticos para o desenvolvimento de lógica de programação utilizando **JavaScript**. A atividade engloba desde operações matemáticas simples e manipulação de dados até estruturas de repetição.

---

## 📋 Sobre a Lista de Exercícios

A atividade está dividida em duas grandes etapas de aprendizado:

1. **Operações Matemáticas e Lógica (Exercícios 1 ao 15):** Scripts focados na entrada de dados, processamento matemático (soma, subtração, médias, porcentagens, cálculo de áreas) e exibição de resultados ao usuário.
2. **Estruturas de Repetição (Exercícios 16 ao 20):** Laços de repetição (`for` / `while`) para iteração de valores, contagens crescentes e decrescentes, além de filtragens condicionais (números pares e ímpares).

---

## 🛠️ Tecnologias Utilizadas

* **JavaScript (ES6+)**: Linguagem principal utilizada para resolver a lógica dos problemas.
* **Node.js / Browser Console**: Ambientes utilizados para a execução e teste dos scripts.

---

## 🚀 Como Executar os Exercícios

Você pode rodar estes códigos de duas maneiras simples:

### Opção 1: No Navegador (Console do Desenvolvedor)
1. Abra qualquer página web no seu navegador (ex: Google Chrome).
2. Pressione `F12` (ou clique com o botão direito e vá em **Inspecionar**) e mude para a aba **Console**.
3. Copie o código do exercício desejado, cole no console e pressione `Enter`.
4. Para as funções interativas (1 ao 15), basta chamar a função no console (Ex: `exercicio01();`).

### Opção 2: Utilizando o Node.js
1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone este repositório ou baixe os arquivos.
3. Abra o terminal na pasta do projeto e execute o comando:
   ```bash
   node nome_do_arquivo.js
